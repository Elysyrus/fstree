Git有三个区域：工作区/暂存区/仓库
五步走 git pull -> git status -> git add -> git commit -> git push

commit message 怎么写
格式：类型: 简短描述
feat    新增功能
fix 修复 bug
refactor    重构代码（功能没变）
docs    改了文档/注释
chore   改了构建配置等杂项

必须理解：fs::path 不是字符串 它知道路径的结构，这就是它强大的地方
fs::path p = "src/main.cpp";
p.filename()   // → "main.cpp"    文件名
p.stem()       // → "main"        不含扩展名
p.extension()  // → ".cpp"        扩展名
p.parent_path()// → "src"         父目录
